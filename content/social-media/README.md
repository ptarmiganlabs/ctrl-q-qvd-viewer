# Social Media Content

This directory contains social media posts and announcements for Ctrl-Q QVD Viewer.

## Files

### LinkedIn Release Announcement

Two versions of the LinkedIn release announcement are available:

1. **linkedin-release-announcement.md** - Full version with metadata and notes
   - Includes post metrics, target audience info, and writing notes
   - Use this for reference and future content creation

2. **linkedin-release-announcement.txt** - Plain text ready-to-post version
   - Clean copy-paste format for LinkedIn
   - No markdown formatting or metadata
   - Simply copy the entire content and paste into LinkedIn

## Usage
