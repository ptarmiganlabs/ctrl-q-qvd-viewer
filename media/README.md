# Media Assets

This directory contains logo images and other media assets used by the extension.

## Required Logo Files

Please save the following logo files to this directory:

1. **ctrl-q-logo.png** - The Ctrl-Q logo (black background with green text version or white background version)
2. **ptarmigan-logo.png** - The Ptarmigan Labs logo

### Recommended Specifications

- Format: PNG with transparency support
- Maximum width: 200px (logos will be automatically scaled)
- Aspect ratio: Original aspect ratio preserved
- Background: Transparent recommended for best compatibility with VS Code themes

## Logo Sources

The logo images are included in the project repository. Use:

- `ctrl-q-logo.png` - Use the version with black or white background depending on preference
- `ptarmigan-logo.png` - The Ptarmigan Labs company logo
