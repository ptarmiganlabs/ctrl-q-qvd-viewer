# Change Log

All notable changes to the "qvd-viewer" extension will be documented in this file.

## [0.0.1] - 2025-10-20

### Added
- Initial release of QVD Viewer extension
- Custom editor for viewing QVD files
- Display QVD file metadata (table name, record count, field count, file size, creator, creation time)
- Display field information (name, type, number format, tags)
- Display data lineage information
- Configurable data preview with customizable row count (default 25 rows)
- Support for QVD files created by Qlik Sense and QlikView
- Built using qvd4js library for QVD file parsing
